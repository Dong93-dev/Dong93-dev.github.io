Project Gallery

NC News

This full stack project produced a Reddit-like website where you can read articles, leave your comments, vote what you like and etc.. The whole project was designed in MVC pattern. The fully-tested backend was developed mainly using node.js, jest, express and knex, which provided a set of RESTful APIs to interact with PSQL. These APIs can be found through my Github link https://github.com/Dong93-dev/be-nc-news. Moreover, the JWT auth was integrated to provide secure access to these APIs. The frontend was developed using React and a little Bootstrap. Feel free to check out my frontend source code https://github.com/Dong93-dev/dong-nc-news-app. The project took about two weeks to complete but I still put many efforts on improvement of UI/UX. Some details were considered, such as optimistic rendering to update the vote number, localStorage to keep the comment in case of accident refreshing, and etc.

Tech stack: JS, Jest, express.js, knex, PSQL, JWT, React, React-Boostrap
